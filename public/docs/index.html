    Introduction

    Essa documentação trás informações sobre como consumir a API da vivily

    Conforme você rola, você verá exemplos de código para trabalhar com a API em diferentes linguagens de programação na área escura à direita (ou como parte do conteúdo no celular). Você pode alterar a linguagem usando as guias no canto superior direito (ou no menu de navegação no canto superior esquerdo no celular).

    Base URL

    http://localhost

    Authenticating requests

    Authenticate requests to this API's endpoints by sending a Authorization header with the value "{YOUR_AUTH_KEY}".

    All authenticated endpoints are marked with a requires authentication badge in the documentation below.

    O token não possui tempo de vida definido. O mesmo ficará ativo até o usuário deslogor ou quando o usuário atingir o número máximo de logins ativos
